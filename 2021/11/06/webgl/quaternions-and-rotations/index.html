定义一个四元数
q = q_0 + q_1i + q_2j + q_3k.

The standard orthonormal basis for R3 is given by three unit vectors
i = (1, 0, 0), j = (0, 1, 0), k = (0, 0, 1).
A quaternion q is defined as the sum of a scalar q0 and a vector q = (q1, q2, q3 ; namely,

四元数加法与乘法

设有两个四元数p和qq=q0+q1i+q2j+q3k.p=p0+p1i+p2j+p3k.则有p+q=(p0+q0)+(p1+q1)i+(p2+q2)j+(p3+q3)k.设有两个四元数p和q\\ q = q_0 + q_1\bm{i} + q_2\bm{j} + q_3\bm{k}.\\ p = p_0 + p_1\bm{i} + p_2\bm{j} + p_3\bm{k}.\\ 则有\\ p + q = (p_0 + q_0) + (p_1 + q_1)i + (p_2 + q_2)j + (p_3 + q_3)k. 设有两个四元数p和qq=q0​+q1​i+q2​j+q3​k.p=p0​+p1​i+p2​j+p3​k.则有p+q=(p0​+q0​)+(p1​+q1​)i+(p2​+q2​)j+(p3​+q3​)k.

四元数的乘法满足以下规定

i2=j2=k2=ijk=−1,ij=k=−ji,jk=i=−kj,ki=j=−ik. \bm{i}^2 = \bm{j}^2 = \bm{k} ^2 = \bm{ijk} = −1,\\ \bm{ij} = \bm{k} = −\bm{ji},\\ \bm{jk} = \bm{i} = −\bm{kj},\\ \bm{ki} = \bm{j} = −\bm{ik}.\\ i2=j2=k2=ijk=−1,ij=k=−ji,jk=i=−kj,ki=j=−ik.

四元数的复共轭、范数与逆

复共轭 Complex Conjugate

设有一个四元数

q=q0+q=q0+q1i+q2j+q3k.q = q_0 + \bm{q} = q_0 + q_1\bm{i} + q_2\bm{j} + q_3\bm{k}.\\ q=q0​+q=q0​+q1​i+q2​j+q3​k.

则其共轭

q∗=q0−q=q0−q1i−q2j−q3k.q^* = q_0 - \bm{q} = q_0 - q_1\bm{i} - q_2\bm{j} - q_3\bm{k}.\\ q∗=q0​−q=q0​−q1​i−q2​j−q3​k.

(pq)∗=q∗p∗(pq)^* = q^* p^* (pq)∗=q∗p∗
