Hexo+Gitee个人博客搭建

|字数总计:490|阅读时长:1分钟|阅读量:

个人博客搭建

Hexo+Gitee

1、环境准备

1.1 git

下载安装git 下载地址 https://git-scm.com/

1.2 node.js

下载安装node.js 下载地址 https://nodejs.org/

2、安装Hexo

Hexo文档 https://hexo.io/zh-cn/docs/

2.1安装Hexo

在安装完成git和node.js以后，即可使用 npm 安装 Hexo。

$ npm install -g hexo-cli

安装完Hexo以后
新建文件夹，在文件夹内右键选择 git bash here,或者按住shfit 右键选择在此处打开CMD或Powershell

2.2新建网站

指令详见文档
https://hexo.io/zh-cn/docs/commands

在命令行中输入命令新建网站

$ hexo init

新建完成后，指定文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
| ├── _drafts
| └── _posts
└── themes

2.3新建文章

$ hexo new "文章标题"

此时会在文件夹中./source/_posts中新建一个md文档，即可在此md文件中写文章

2.4生成静态文件

$ hexo g

2.5启动服务器

在命令行中输入命令

$ hexo s

返回结果


INFO Validating config

INFO Start processing

INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

然后在浏览器中输入地址 http://localhost:4000 即可访问预览

3.部署到Gitee

3.1新建仓库

仓库名需要与账号名一致

3.2安装git部署插件

npm indtall hexo-depolyer-git --save

修改项目配置文件 _config.yml
打开_config.yml文件，在文件最下方配置新建仓库的地址

# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
type: 'git'
repo: https://gitee.com/xiangxun1/xiangxun1.git
branch: master type中写git，repo写新建仓库地址，branch填写master；

然后再输入命令

hexo d

即可将本地的网站部署到gitee新建的仓库中，在本地的文件夹中也会新建一个public文件夹

注：在输入hexo d 以后会提示登录gitee的账号

3.3 Gitee Pages

开启Gitee Pages静态网页托管服务

注：需要先通过实名认证

完成以后就可以在 https://xiangxun1.gitee.io 中访问了
